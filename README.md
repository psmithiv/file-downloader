# file-downloader
Command line based file download tool.

# Project Setup
* npm i -g typescript
* npm i -g ts-node
* npm i -g nexe
* npm install

# Style Guild
* https://github.com/Platypi/style_typescript

# Run Project (development)
* ts-node cli

# Build Project
* npm run build

# Usage
* downloader.exe -h
